## How to Run

1. Clone the repository

```
git clone https://github.com/yourusername/airbnb-clone-fullstack.git
```

2. Open the project folder

```
cd airbnb-clone-fullstack
```

3. Install dependencies

```
npm install
```

4. Start the server

```
npm start
```

5. Open in browser

```
http://localhost:3000
